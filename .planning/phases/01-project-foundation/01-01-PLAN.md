# Plan 01-01: Gradle Project Setup

## Objective

Initialize the BioBell Android project with a production-ready Gradle configuration using Kotlin 2.0, AGP 8.3+, and a Version Catalog for all dependencies.

## Context

- App ID: `com.biobell.android`
- Min SDK: 31 (Android 12 — required for Material You dynamic color)
- Target SDK: 34 (Android 14)
- Compile SDK: 34
- Kotlin: 2.0.x
- AGP: 8.3.x

## Tasks

### Task 1: Create settings.gradle.kts

Create `settings.gradle.kts` with:
- `pluginManagement` block pointing to Google and Maven Central repos
- `dependencyResolutionManagement` block with `libs.versions.toml` version catalog
- Project name: `BioBell`
- Include `:app` module

### Task 3: Create root build.gradle.kts

```kotlin
// Top-level build file
plugins {
    alias(libs.plugins.android.application) apply false
    alias(libs.plugins.kotlin.android) apply false
    alias(libs.plugins.kotlin.compose) apply false
    alias(libs.plugins.hilt) apply false
    alias(libs.plugins.ksp) apply false
}
```

### Task 4: Create app/build.gradle.kts

Full app module build file with:
- All plugins (android, kotlin, compose, hilt, ksp)
- `compileSdk = 34`, `minSdk = 31`, `targetSdk = 34`
- `applicationId = "com.biobell.android"`
- `versionCode = 1`, `versionName = "1.0.0"`
- `buildFeatures { compose = true }`
- All dependencies from libs catalog
- `compileOptions { sourceCompatibility/targetCompatibility = JavaVersion.VERSION_17 }`
- `kotlinOptions { jvmTarget = "17" }`
